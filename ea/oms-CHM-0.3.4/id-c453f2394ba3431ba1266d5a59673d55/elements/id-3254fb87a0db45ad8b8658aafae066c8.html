C7.2.2-5 ()
C7.2.2-5 ประเด็นปัญหาและสิ่งที่ต้องปรับปรุงในอนาคต - E/O update ข้อมูลในระบบ OMS อาทิ สถานะเหตุการณ์ สถานะการแก้ไขไฟฟ้า เวลาจ่ายไฟกลับคืน ล่าช้าในบางเหตุการณ์ เนื่องจากต้องใช้เวลาในการตรวจสอบสภาพหน้างาน - ยังต้องใช้วิธีการตรวจสอบข้อมูลจากหลายแหล่ง เช่น Open Chat- ระบบ SCADA ยังไม่สามารถตรวจพบไฟฟ้าดับได้ทั้งเครือข่ายไฟฟ้า การปรับปรุงเพื่อเพิ่มประสิทธิภาพ - มีการกาหนดตัวชี้วัดการดาเนินงานให้ E/O ต้องดาเนินการ Update เหตุการณ์ไฟฟ้าดับในระบบ OMS - ปรับปรุงวิธีการบริหารจัดการ Case ของ Agent กรณียังไม่สามารถตรวจสอบเหตุการณ์ไฟฟ้าขัดข้องได้ - ปรับปรุงระบบปัจจุบัน (อาทิ ระบบ GIS ระบบ OMS) ให้สามารถแสดงข้อมูลพื้นที่ที่ได้รับผลกระทบจากเหตุการณ์ไฟฟ้าดับ
