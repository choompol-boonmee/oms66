Apply switchingchange to data model(Automatic)(Backup: HV/MV Dispatcher): 860 ()
ส่งข้อมูลการทำสวิชชิ่งเข้าสู่ data model SCADA/eRespond interface จะทำการปรับปรุงข้อมูลสถานะการจ่ายไฟใน eRespond ตามสภาพความเป็นจริงในขณะนั้น ระบบสำรอง: ตามการร้องขอการเปลี่ยนแปลงกับ กฟภ. ระบบ eRespond จะอนุญาติให้พนักงานสั่งการระบบสายส่งหรือระบบจำหน่ายแรงสูงทำการปลดสับอุปกรณ์ไฟฟ้าในแบบจำลองเครือข่ายระบบไฟฟ้าใน eRespond ได้โดยตรงไม่จำกัดว่าเป็นเขตที่มีระบบ SCADA ติดตั้งหรือไม่ โดยกระบวนการทางธุรกิจนี้จะต้องดำเนินการโดยพนักงานสั่งการระบบสายส่งหรือระบบจำหน่ายแรงสูงในกรณีที่สัญญาณข้อมูลการเชื่อมต่อระหว่างระบบ SCADA และ eRespond ล่มเท่านั้น
