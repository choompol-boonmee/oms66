3.3.1 โครงการกำกับดูแลและ บริหารงานจัดซื้อพร้อมติดตั้ง โครงสร้างพื้นฐานแบบรวมศูนย์ (ITiDC) ()
สำหรับเครื่องคอมพิวเตอร์แม่ข่าย ของการไฟฟ้าส่วนภูมิภาค มีการบริหารจัดการแบบรวมศูนย์ จากการประชุมหารือ กับทาง ฝสท พบว่า มีการจัดเตรียมไว้ในลักษณะโครงสร้างพื้นฐานแบบรวมศูนย์ กล่าวคือ เตรียมหน่วยประมวลกลางเสมือน หรือ vCPU ไว้เพื่อให้บริการระบบซอฟต์แวร์ต่างๆ ที่จำเป็นต้องใช้เครื่องคอมพิวเตอร์แม่ข่าย โดยได้เตรียมเครื่องคอมพิวเตอร์แม่ข่าย ที่เป็นฮาร์ดแวร์ไว้ทั้งที่ ศูนย์ข้อมูลกลางของ PEA ที่สำนักงานใหญ่ (PEA iDC) และ ที่ศูนย์ข้อมูลสำรองของ PEA (PEA DR Site) ที่ละประมาณ 1300 แกน (1300 core) ซึ่งสถานะปัจจุบัน มีแกนเหลืออยู่ไม่มากแล้ว ประมาณ 200 แกน ส่วนหน่วยความจำหลักได้เตรียมไว้ประมาณ 270 TB ในการออกแบบฮาร์ดแวร์สำหรับเครื่องคอมพิวเตอร์แม่ข่าย ซึ่งใช้จำนวนแกน เป็นจำนวนมาก จึงน่าจะเหมาะสมที่จะให้โครงการจัดหาเครื่องคอมพิวเตอร์แม่ข่ายเอง ภายใต้งบประมาณโครงการ สำหรับเครื่องคอมพิวเตอร์สำหรับผู้ใช้งาน (OMS Workstation) ซึ่งจำเป็นต้องใช้เครื่องคอมพิวเตอร์ที่มีข้อกำหนดทางเทคนิกที่ค่อนข้างสูงมาก ทั้งทางส่วนที่เป็นหน่วยประมวลกลางก็ใช้ที่ประสิทธิภาพมากกว่า ทั้งในส่วนที่เป็นจอมอนิเตอร์ซึ่งมีขนาดค่อนข้างใหญ่และมีจำนวนหลายจอ และที่สำคัญคือต้องใช้การ์ดจอ (Graphic Card) ที่มีประสิทธิภาพสูงกว่า เครื่อง Desktop PC ที่เป็นข้อกำหนดกลางของ PEA หรือ ข้อกำหนดของประเทศ และเมื่อพิจารณาความเป็นไปได้ในการใช้เครื่อง VDI (Virtual Desktop Infrastructure) ที่ PEA มีอยู่ พบว่า ข้อกำหนดของเครื่อง VDI ดังกล่าว ดังนั้นภายใต้โครงการจัดหาระบบบริหารไฟฟ้าขัดข้องในโครงการนี้ จำเป็นต้องคำนึงถึงเครื่องคอม
