เหตุการณ์นั้นจะสร้างใบสั่งงานอัตโนมัติหรือไม่ eRespond จะตรวจสอบอย่างอัตโนมัติว่าเหตุการณ์ใหม่นั้น จะสร้างใบสั่งงานอัตโนมัติหรือไม่ ในกรณีที่ใบสั่งงานไม่ได้ถูกสร้างขึ้นโดยอัตโนมัติจากกระบวนการนี้พนักงานสั่งงานแก้ไขกระแสไฟฟ้าขัดข้องประจำ กฟฟ. จะสามารถจัดการเหตุการณ์นั้นใน eRespond และสร้างใบสั่งงานขึ้นถ้าต้องการด้วยมือ และสามารถดำเนินการต่อไปยังกระบวนการหมายเลข 250. คุณสมบัติในการสร้างใบสั่งงานอัตโนมัตินั้นจะมีเฉพาะประเภทเหตุการณ์บำรุงรักษา (ที่ไม่ใช่ไฟฟ้าขัดข้อง) และมีสภาพเหตุการณ์ที่เหมาะสม ส่วนประเภทเหตุการณ์อื่นๆ นั้น จำเป็นที่จะต้องให้พนักงานสั่งงานแก้ไขกระแสไฟฟ้าขัดข้องประจำ กฟฟ. ดำเนินการพิจารณาเพื่อสร้างใบสั่งงานด้วยตนเอง หมายเหตุ: พนักงานสั่งงานแก้ไขกระแสไฟฟ้าขัดข้องประจำ กฟฟ. ไม่สามารถสร้างใบสั่งงานสำหรับเหตุการณ์รับแจ้งปัญหาที่ไม่ได้อ้างอิงสถานที่ใช้ไฟหรือไม่มีอุปกรณ์ไฟฟ้า (มิเตอร์) ที่ระบุได้ในเหตุการณ์ที่แจ้งมา
