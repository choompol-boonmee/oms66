ปรับสถานะเหตุการณ์แผนไฟดับ(ประเภทPO)เป็นสถานะอนุมัติ(AP)พร้อมแนบเอกสารขั้นตอน สวิชชิ่งเข้าใบสั่งงานดับไฟ ประเภท(PH,PV,PL) /หรือ หากมีการทําสวิชชิ่งย้ายโหลดดับไฟมากกว่า 1 กฟฟ. ADDC WorkDispatcher จะสร้างใบสั่งงาน เพิ่มและแจ้งให้ E/O Work Dispatcher
พิมพ์รายชื่อผู้ใช้ไฟที่มีผลกระทบเกี่ยวกับ แผนดับไฟนําเสนอผจก.หรือผู้ปฏิบัติงาน แทนให้ส่วนเกี่ยวข้องประกาศดับไฟ
