ในรายงานฉบับนี้ จะใช้วิธีการ สถาปัตยกรรมองค์กร หรือ EA (Enterprise Architecture) ในการจัดทำภาพและสัญญลักษณ์ในการอธิบาย ทั้งนี้เพื่อความชัดเจนในความหมาย และเพื่อเป็นเอกภาพในการจัดทำเอกสาร และเพื่อให้การอธิบายความมีความเป็นสากลได้ได้มาตรฐาน ในรายงานฉบับนี้ ใช้สัญญลักษณ์และวิธีการ ตามมาตรฐานสากล ArchiMate เวอร์ชั่น 3.2 ซึ่งเป็นมาตรฐานที่ประกาศโดยองค์กรสากล Open Group สอดคล้องกับมาตรฐาน TOGFA โดยสามารถอธิบายสัญญลักษณ์ภาพโดยสังเขป ได้ดังต่อไปนี้
