15-JD-กองวิศวะ ()
JD - กองวิศวกรรมและวางแผน
1. งานวิเคราะห์ วางแผนระบบไฟฟ้าและการปรับปรุงระบบไฟฟ้า
4. งานจัดทำ ปรับปรุงแผนที่ระบบไฟฟ้า ข้อมูลสารสนเทศระบบไฟฟ้า และพัฒนาApplication ที่เกี่ยวข้อง
7. ควบคุม ดูแล มาตรฐานงานก่อสร้างระบบไฟฟ้า การทดสอบอุปกรณ์ไฟฟ้า และวิเคราะห์แก้ไขปัญหาด้านวิศวกรรม
3. จัดการงานซื้อขายไฟฟ้าระหว่างประเทศและการเชื่อมโยงระบบของผู้ผลิตไฟฟ้าขนาดเล็ก(SPP)ขนาดเล็กมาก(VSPP)
5. การจัดการการใช้พลังงานไฟฟ้าอย่างมีประสิทธิภาพและอนุรักษ์พลังงาน
2. สำรวจ ออกแบบ ประมาณการสถานีไฟฟ้าชั่วคราว สายส่ง และระบบจำหน่าย
6. งานความปลอดภัยอาชีวอนามัยและสิ่งแวดล้อมในที่ทำงาน
JD - แผนกแผนที่ระบบไฟฟ้า
1. ตรวจสอบ แนะนำ การปรับปรุงข้อมูลงานสารสนเทศระบบไฟฟ้าทางภูมิศาสตร์(GIS)
4. ติดตาม ตรวจสอบ ประเมินผล การใช้งานโปรแกรมประยุกต์ต่าง ๆ(Application)พร้อมปรับปรุงพัฒนาเพิ่มประสิทธิภาพ เพื่อสนับสนุนการใช้งานให้เกิดประโยชน์สูงสุด
3. ให้บริการข้อมูลงานสารสนเทศระบบไฟฟ้าทางภูมิศาสตร์(GIS) ทั้งหน่วยงานภายในและภายนอก กฟภ.
5. ดูแล ปรับปรุง บำรุงรักษาระบบคอมพิวเตอร์ และระบบสารสนเทศระบบไฟฟ้าทางภูมิศาสตร์(GIS) และสนับสนุนข้อมูลที่ใช้ในการวิเคราะห์และตัดสินใจของผู้บริหาร
2. ตรวจสอบ คุณภาพข้อมูลสารสนเทศระบบไฟฟ้าทางภูมิศาสตร์(GIS) และประสานงานปรับปรุงแก้ไข นำเข้าข้อมูลให้ถูกต้องเป็นปัจจุบัน
JD - กองวิศวกรรมและวางแผน 2. สำรวจ ออกแบบ ประมาณการสถานีไฟฟ้าชั่วคราว สายส่ง และระบบจำหน่าย
JD - กองวิศวกรรมและวางแผน 4. งานจัดทำ ปรับปรุงแผนที่ระบบไฟฟ้า ข้อมูลสารสนเทศระบบไฟฟ้า และพัฒนาApplication ที่เกี่ยวข้อง
JD - กองวิศวกรรมและวางแผน 7. ควบคุม ดูแล มาตรฐานงานก่อสร้างระบบไฟฟ้า การทดสอบอุปกรณ์ไฟฟ้า และวิเคราะห์แก้ไขปัญหาด้านวิศวกรรม
JD - กองวิศวกรรมและวางแผน 3. จัดการงานซื้อขายไฟฟ้าระหว่างประเทศและการเชื่อมโยงระบบของผู้ผลิตไฟฟ้าขนาดเล็ก(SPP)ขนาดเล็กมาก(VSPP)
JD - กองวิศวกรรมและวางแผน 5. การจัดการการใช้พลังงานไฟฟ้าอย่างมีประสิทธิภาพและอนุรักษ์พลังงาน
JD - กองวิศวกรรมและวางแผน 1. งานวิเคราะห์ วางแผนระบบไฟฟ้าและการปรับปรุงระบบไฟฟ้า
JD - กองวิศวกรรมและวางแผน 6. งานความปลอดภัยอาชีวอนามัยและสิ่งแวดล้อมในที่ทำงาน
JD - แผนกแผนที่ระบบไฟฟ้า 1. ตรวจสอบ แนะนำ การปรับปรุงข้อมูลงานสารสนเทศระบบไฟฟ้าทางภูมิศาสตร์(GIS)
JD - แผนกแผนที่ระบบไฟฟ้า 4. ติดตาม ตรวจสอบ ประเมินผล การใช้งานโปรแกรมประยุกต์ต่าง ๆ(Application)พร้อมปรับปรุงพัฒนาเพิ่มประสิทธิภาพ เพื่อสนับสนุนการใช้งานให้เกิดประโยชน์สูงสุด
JD - แผนกแผนที่ระบบไฟฟ้า 3. ให้บริการข้อมูลงานสารสนเทศระบบไฟฟ้าทางภูมิศาสตร์(GIS) ทั้งหน่วยงานภายในและภายนอก กฟภ.
JD - แผนกแผนที่ระบบไฟฟ้า 2. ตรวจสอบ คุณภาพข้อมูลสารสนเทศระบบไฟฟ้าทางภูมิศาสตร์(GIS) และประสานงานปรับปรุงแก้ไข นำเข้าข้อมูลให้ถูกต้องเป็นปัจจุบัน
JD - แผนกแผนที่ระบบไฟฟ้า 5. ดูแล ปรับปรุง บำรุงรักษาระบบคอมพิวเตอร์ และระบบสารสนเทศระบบไฟฟ้าทางภูมิศาสตร์(GIS) และสนับสนุนข้อมูลที่ใช้ในการวิเคราะห์และตัดสินใจของผู้บริหาร
