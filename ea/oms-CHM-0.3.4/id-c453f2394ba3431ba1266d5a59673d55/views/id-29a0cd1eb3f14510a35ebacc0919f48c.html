JD - แผนกพัฒนาระบบงานด้านวิศวกรรม
1. ดูแล วิเคราะห์ ออกแบบ พัฒนา สนับสนุน และบำรุงรักษา ระบบงานสารสนเทศด้านวิศวกรรม
4. ให้คำปรึกษา แนะนำ และแก้ปัญหาการใช้ระบบสารสนเทศในทางเทคนิคที่อยู่ในความดูแล
3. ประสานงานในการให้ข้อมูลและเข้าร่วมปฏิบัติงานในโครงการตามแผนปฏิบัติการ ดิจิทัลของ การไฟฟ้าส่วนภูมิภาค ในส่วนที่เกี่ยวข้องกับระบบงานที่อยู่ในความดูแล
5. วางแผน ออกแบบ พัฒนา ระบบงานและปรับปรุงกระบวนงานเพื่อเพิ่มประสิทธิ ภาพและรองรับความต้องการด้านดิจิทัลขององค์กร
2. วางแผนและจัดการระบบงานสารสนเทศที่อยู่ในความดูแล ให้ดำเนินงานไปอย่างมีประสิทธิภาพ
6. ทดสอบระบบงานที่ออกแบบและพัฒนา และจัดทำเอกสารคู่มือระบบ คู่มือการใช้งานของระบบงานที่พัฒนา/ปรับปรุง ตลอดจนเอกสารที่เกี่ยวข้อง
