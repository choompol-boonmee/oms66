23 ()
นโยบาย 23) ผู้บังคับบัญชาชั้นต้นขึ้นไปต้องกำหนดบุคคลดูแลควบคุมการใช้งานและรับผิดชอบทรัพย์สิน สารสนเทศให้ชัดเจน ตามระเบียบ คำสั่ง หลักเกณฑ์ และหรือแนวทางปฏิบัติที่เกี่ยวกับความมั่นคงปลอดภัย สารสนเทศของ กฟภ. ที่ประกาศใช้ในปัจจุบัน แนวทางปฏิบัติ 23.1 หัวหน้าหน่วยงานตั้งแต่ผู้อานวยการกองขึ้นไปควรกำหนดบุคคลดูแลควบคุมการใช้งาน และรับผิด ชอบทรัพย์สินสารสนเทศให้ชัดเจนในแต่ละรายการโดยจัดหมวดหมู่ ของทรัพย์สินที่ตนเองถือครองตามหมวดต่าง ๆ เช่น หมวดอุปกรณ์คอมพิวเตอร์ หมวดซอฟต์แวร์ หมวดอุปกรณ์สื่อสารและเครือข่าย หมวดข้อมูลและเอกสาร เป็นต้น 23.2 โดยบุคคลใดครอบครองอะไรก็ให้บุคคลนั้นรับผิดชอบทรัพย์สิน สารสนเทศนั้น ส่วนทรัพย์สินสารสนเทศส่วนกลางให้ผู้อานวยการกองกำหนดบุคคลดูแล
