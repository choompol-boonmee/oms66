กระบวนการทำเรื่องคืน/เบิกวัสดุอุปกรณ์ กระบวนการนี้จะเป็นการดำเนินเรื่องในการเบิกวัสดุอุปกรณ์เพื่อทดแทนของให้กับของในรถที่ถูกใช้ไป, การทำเรื่องส่งคืนคลังสำหรับวัสดุอุปกรณ์ที่เบิกไปแต่ไม่ได้ใช้, การทำเรื่องคืนของที่เสียหายอันเนื่องมาจากการเกิดกระแสไฟฟ้าขัดข้อง (การปรับปรุงมูลค่าของบัญชีทรัพย์สิน) และเคลียร์บัญชีวัสดุอุปกรณ์ที่ได้ถูกเบิกใช้งานตามใบสั่งงาน
