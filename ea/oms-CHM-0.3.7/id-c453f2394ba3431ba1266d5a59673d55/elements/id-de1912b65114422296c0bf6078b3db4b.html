3.5.1 ปรับปรุงกระบวนงานให้มีการสลับจอการทำงานน้อยลง โดยการเชื่อมโยงระบบมากขึ้น ()
ปัจจุบันเจ้าหน้าที่ E/O จำเป็นต้องใช้ซอฟต์แวร์หลายตัว ทำงานแตกต่างกันไป ที่สำคัญได้แก่ ระบบ GIS สำหรับค้นหาตำแหน่งที่เกิดเหตุไฟฟ้าขัดข้อง ระบบ eResponse เพื่อบันทึกเหตุการณ์ เปิดใบสั่งงาน บันทึกการสั่งงาน ระบบ SAP เพื่อบันทึกข้อมูลการทำงาน และ ระบบ MWM (Mobile Workforce Management) ในการประสานเจ้าหน้าที่หน้างาน ระบบดังกล่าวยังไม่สามารถเชื่อมโยงกันได้อย่างมีประสิทธิภาพเท่าที่ควร บางเวลาการเชื่อมโยงก็หลุด คล้ายกับว่าไม่สามารถส่งข้อมูลข้ามระบบได้อย่างมีประสิทธิภาพ ถ้ามีการจัดหาและพัฒนาระบบบริหารไฟฟ้าขัดข้องใหม่ มีการพัฒนาการเชื่อมโยงได้อย่างมีประสิทธิภาพ ก็จะสามารถลดการทำงานจากหลายหน้าจอ เหลือเพียงหน้าจอเดียว ทำให้สามารถปรับปรุงกระบวนงานให้ทำงานได้ง่ายขึ้น รวดเร็วขึ้น
