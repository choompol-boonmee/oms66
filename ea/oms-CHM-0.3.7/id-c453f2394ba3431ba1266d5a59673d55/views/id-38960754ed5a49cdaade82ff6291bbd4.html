โครงการเพิ่มประสิทธิภาพระบบศูนย์สั่งการจ่ายไฟ หรือ โครงการพัฒนาระบบ SCADA เฟส 3 เป็นโครงการที่พัฒนาขึ้นเพื่อเปลี่ยนซอฟต์แวร์สำหรับศูนย์สั่งการจ่ายไฟ ให้เป็นซอฟต์แวร์ผลิตภัณฑ์ใหม่ คือผลิตภัณฑ์ของ Schneider Electric สถาปัตยกรรมระบบซอฟต์แวร์ของผลิตภัฑณ์นี้ สามารถอธิบายได้ด้วยภาพข้างล่างต่อไปนี้
