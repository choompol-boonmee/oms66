16-JD-กองปฏิบัติ-1 ()
JD - แผนกควบคุมการจ่ายไฟ
1. งานปฏิบัติการ (Operate) อุปกรณ์ในสายส่ง สถานีไฟฟ้าและระบบจำหน่ายแรงสูงโดยระบบSCADA
4. ตรวจสอบ ควบคุม รักษาแรงดันไฟฟ้าระบบ 22 kVขึ้นไปให้อยู่ในเกณฑ์มาตรฐาน
7. จัดทำแผนปลดโหลด (Load Shedding Scheme) และแผนการจ่ายไฟฟ้าคืน(Restoration Plan)
3. สั่งการ และสั่งงานในระบบบริหารไฟฟ้าขัดข้อง และบันทึกเหตุการณ์ไฟฟ้าขัดข้องในระบบสายส่ง สถานีไฟฟ้า และระบบจำหน่ายแรงสูง
5. จัดทำและปรับปรุง Switching Diagram ให้เป็นปัจจุบัน
2. ประสานงานการจ่ายไฟกับหน่วยงานภายในและภายนอก และ ควบคุมการรับ-จ่ายไฟจากผู้ผลิตไฟฟ้าขนาดเล็ก(SPP)และขนาดเล็กมาก(VSPP)
6. จัดทำและตรวจสอบ Switching Order สำหรับการดับไฟฟ้าแบบมีแผนงานและกรณีฉุกเฉิน
8. ติดตาม เฝ้ามองระบบแจ้งเตือนภัยผู้บุกรุกและระบบแจ้งเตือนอัคคีภัย ภายในสถานีไฟฟ้า
