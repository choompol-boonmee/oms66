C6.1.4.1 งานบริหารจัดการเหตุการณ์ ระบบสายส่ง สาเหตุภายใน ของ กฟภ.
C6.1.4.1-1 ตรวจพบปัญหา ในระบบจำหน่าย
C6.1.4.1-2 HV/MV/Dispatcher พิจารณาย้าย/ถ่ายเท โหลดและมอบหมายงาน ค้นหาสาเหตุ (ย้ายโหลดและจ่ายไฟ ในสายส่งที่ยังใช้งานได้)
C6.1.4.1-3 สร้างเหตุการณ์ ไฟฟ้าขัดข้องอัตโนมัติ ในระบบ OMS
C6.1.4.1-4 ตรวจสอบและสร้าง เหตุการณ์ ไฟฟ้าขัดข้อง บนระบบ OMS
C6.1.4.1-5 แจ้งขอใบสั่งงานจาก SMC
C6.1.4.1-6 สร้างใบสั่งงาน เพื่อให้ชุดปฏิบัติการ ระบบไฟฟ้า เข้าตรวจสอบ และค้นหาสาเหตุ
C6.1.4.1-7 สร้างและมอบหมายใบสั่งงาน เพื่อให้ชุดปฏิบัติการ ระบบไฟฟ้า เข้าตรวจสอบ และค้นหาสาเหตุ
C6.1.4.1-8 รับงาน ออกเดินทาง ค้นหาสาเหตุไฟฟ้าขัดข้อง เมื่อพบสาเหตุ ตรวจสอบรายงานสาเหตุ สภาพหน้างาน พิจารณาแนวทางแก้ไข และประมาณการ ระยะเวลาแก้ไข หรือ ประมาณการเวลาจ่ายไฟกลับคืน
