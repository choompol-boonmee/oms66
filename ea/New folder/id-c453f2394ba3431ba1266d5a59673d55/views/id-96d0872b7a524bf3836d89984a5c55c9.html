16-JD-กองปฏิบัติ-2 ()
JD - แผนกวิเคราะห์และวางแผนการจ่ายไฟ
1. วิเคราะห์และวางแผนการเปลี่ยนแปลงการจ่ายไฟในสายส่งและระบบจำหน่ายแรงสูง
4. ตรวจสอบและปรับปรุงฐานข้อมูล (Database) สำหรับจัดทำค่าดัชนีความเชื่อถือได้ของระบบไฟฟ้า
3. สำรวจ ออกแบบประมาณการ ย้ายติดตั้ง รื้อถอน จัดสรร อุปกรณ์ควบคุม อุปกรณ์ป้องกันอุปกรณ์ตัดตอน และอุปกรณ์เพิ่มประสิทธิภาพ (AVR, SVR,Capacitor)ในสายส่งและระบบจำหน่ายแรงสูง
5. พิจารณากำหนดค่าการทำงานของอุปกรณ์เพิ่มประสิทธิภาพ (AVR, SVR, SwitchedCapacitor)
2. ประมวลผลและวิเคราะห์ความเชื่อถือได้ของระบบไฟฟ้าและไฟฟ้าขัดข้อง
6. งานวิเคราะห์และประเมินค่าหน่วยสูญเสียในระบบไฟฟ้าด้านเทคนิค (TechnicalLoss)
JD - แผนกวิเคราะห์และวางแผนการจ่ายไฟ 2. ประมวลผลและวิเคราะห์ความเชื่อถือได้ของระบบไฟฟ้าและไฟฟ้าขัดข้อง

JD - แผนกวิเคราะห์และวางแผนการจ่ายไฟ 3. สำรวจ ออกแบบประมาณการ ย้ายติดตั้ง รื้อถอน จัดสรร อุปกรณ์ควบคุม อุปกรณ์ป้องกันอุปกรณ์ตัดตอน และอุปกรณ์เพิ่มประสิทธิภาพ (AVR, SVR,Capacitor)ในสายส่งและระบบจำหน่ายแรงสูง
