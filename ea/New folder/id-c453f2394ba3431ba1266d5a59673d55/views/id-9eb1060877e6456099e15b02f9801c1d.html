02-ASIS-FLOW ()
เนื่องจากระบบ OMS มีการเชื่อมโยงกับระบบซอฟต์แวร์อื่นจำนวนหลายระบบ ดังนั้นการที่จัดหา พัฒนา ติดตั้ง ระบบ OMS ใหม่ จึงไม่ใช่เป็นงานเดียว กล่าวคือ ไม่ใช่เพียงแต่จัดหาซอฟต์แวรใหม่ มาทดแทนซอฟต์แวร์เดิมเท่านั้น ยังหมายถึงการพัฒนาเชื่อมโยงให้สามารถทำงานเข้ากันได้กับระบบอื่นๆ ด้วย นอกจากนี้ ปัจจุบัน PEA กำลังพัฒนาระบบอื่นๆ ที่มีการเชื่อมโยงกับระบบ OMS อีกด้วย ทำให้มีความจำเป็นต้องประเมินความเกี่ยวข้องกับระบบซอฟต์แวร์ ทุกระบบอย่างละเอียด ภาพต่อไปนี้แสดงการเชื่อมโยงระบบของ ระบบ OMS กับระบบอื่น โดยปัจจุบัน ใช้ผลิตภัณฑ์ซอฟต์แวร์ eResponse เวอร์ชั่น 16 ดังแสดงในภาพต่อไปนี้
