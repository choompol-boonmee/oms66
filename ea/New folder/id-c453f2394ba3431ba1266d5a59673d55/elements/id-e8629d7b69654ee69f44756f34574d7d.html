JD - ศูนย์สั่งการระบบไฟฟ้า
2. ควบคุม ดูแล และบำรุงรักษาอุปกรณ์ระบบศูนย์สั่งการจ่ายไฟ ทั้งฮาร์ดแวร์และซอฟต์แวร์ รวมถึงฐานข้อมูลและส่วนแสดงผลและระบบสื่อสารสำหรับระบบศูนย์เพื่อให้ระบบพร้อมใช้งานอยู่เสมอ ตลอดจนให้การสนับสนุนในการบำรุงรักษา และการจัดหาอุปกรณ์ระบบศูนย์สั่งการจ่ายไฟ และระบบสื่อสารสำหรับระบบศูนย์ฯ ให้แก่ กฟฟ. เขต
