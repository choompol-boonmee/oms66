ประสานงานติดต่อชุดแก้ไฟที่มอบหมายงานออกไป โอเปอเรตอุปกรณ(สวิชชิ่งย้ายโหลด)ย้ายโหลดกลับคืน และปรับสถานะงานโอเปอเรตุปกรณ์(สวิชชิ่งย้ายโหลด) เป็น จ่ายงานเมื่อชุดแก้ไฟถึงหน้างานปรับสถานะเป็น อยู่หน้างาน ในใบสั่งงานประเภท(PH,PV,PL,)
ประสานงานการสั่งการผ่านวิทยุสื่อสารและปรับสถานะ อุปกรณ์ (ปลด/สับ)สวิชชิ่งย้ายโหลดกลับคืนตามการ โอเปอเรตอุปกรณ์จริง เมื่อย้ายโหลด จ่ายไฟ กลับคืน แล้วเสร็จแจ้งใหE/OWork Dispatcher ,หัวหน้าเวรฯ กฟส.,หัวหน้าชุดขอดับไฟทราบ
