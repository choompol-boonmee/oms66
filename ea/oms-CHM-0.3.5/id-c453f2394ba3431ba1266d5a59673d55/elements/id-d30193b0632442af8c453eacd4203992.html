Work Dispatching ()
อุปกรณ์ป้องกันที่สามารถควบคุมระยะไกลได้ทำงานเปิดวงจร อุปกรณ์ป้องกันที่สามารถควบคุมระยะไกลจะสามารถส่งข้อมูลมาให้ SCADA เพื่อรายงานการทำงานเปิดวงจร แต่ในบางครั้งระบบอาจติดต่อไปยัง SCADA ไม่ได้จึงจำเป็นที่ชุดแก้ไฟ/พนักงานประจำสถานีไฟฟ้า/พนักงานสั่งงานแก้ไขกระแสไฟฟ้าขัดข้องประจำ กฟฟ. จะต้องส่งข้อมูลไปให้พนักงานสั่งการระบบสายส่งหรือระบบจำหน่ายแรงสูง
