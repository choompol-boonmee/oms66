ปรับสถานะเหตุการณ์แผนดับไฟ(ประเภทPO)เป็นสถานะ อยู่ระหว่างดําเนินการ(IP)
ประสานงานการสั่งการผ่านวิทยุสื่อสารและปรับสถานะอุปกรณ์ (ปลด/สับ)สวิชชิ่งดับไฟตามการโอเปอเรตอุปกรณ์จริงเมื่อย้าย โหลด ดับไฟ แล้วเสร็จแจ้งให้E/O Work Dispatcher กฟจ.,กฟอ.,หัว หน้าเวรฯ กฟส.,หัวหน้าชุด ขอดับไฟทราบบริเวณพื นที่ดับไฟปฏิบัติงาน
