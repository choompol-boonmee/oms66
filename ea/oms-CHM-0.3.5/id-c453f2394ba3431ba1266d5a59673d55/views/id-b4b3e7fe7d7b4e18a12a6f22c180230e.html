ในปัจจุบันมีการใช้มิเตอร์แบบสมาร์ทมิเตอร์มากขึ้น ระบบเชื่อมโยงสมาร์ทมิเตอร์ เรียกรวมๆ ว่า AMI (Advanced Meter Infrastructure) โดยเป็นระบบที่ใช้มิเตอร์แบบใหม่ มีระบบคอมพิวเตอร์และระบบสื่อสารอยู่ในตัว สามารถติดต่อสื่อสารกับส่วนกลางเพื่อรับส่งข้อมูลระหว่างกันทำให้ ส่วนกลางสามารถรู้ได้ตลอดเวลาว่า มีเหตุการณ์ไฟฟ้าขัดข้องหรือไม่ ภาพข้างล่างต่อไปนี้ แสดงให้เห็นถึงการเชื่อมโยงระบบ AMI กับระบบบริหารไฟฟ้าขัดข้อง OMS และระบบ ADMS
