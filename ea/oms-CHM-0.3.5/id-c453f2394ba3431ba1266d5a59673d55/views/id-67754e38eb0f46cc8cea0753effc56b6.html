01-GMDM ()
ในปัจจุบันระบบซอฟต์แวร์ที่สนับสนุนยูทิลิตี้ มีแนวโน้มที่จะพัฒนาผลิตภัณฑ์ของตนให้สามารถเชื่อมโยงแลกเปลี่ยนข้อมูลกับผลิตภัณฑ์อื่นได้อย่างมีประสิทธิภาพโดยใช้มาตรฐานสากล หลายปีนี้ผู้ผลิตซอฟต์แวร์เหล่านั้นได้ร่วมกันพัฒนาแลกเปลี่ยนข้อมูลระหว่างกัน โดยมีการทดสอบการแลกเปลี่ยนข้อมูลระหว่างกันด้วยมาตรฐานสากลที่เรียกว่ามาตรฐาน CIM ดังแสดงในภาพข้างล่าง ความเคลื่อนของการพัฒนาซอฟต์แวร์นี้เรียกว่า GMDM หรือ Grid Model Data Management ซึ่งมีหลายผู้ผลิตรายใหญ่ในโลกเข้าร่วม
