GA5 : SCADA (SRTU and RTU) Link Down
GA6 : SCADA ยังไม่สามารถเชื่อมโยงข้อมูล วงจรระบบไฟฟ้ากับระบบ GIS
GA7 : SCADA ข้อมูลวงจร ระบบไฟฟ้าไม่เป็นปัจจุบันเนื่องจาก ต้องใช้บุคลากรและเวลาในการดำเนินงาน
GA8 : ระบบ SCADA ส่งข้อมูลสัญญาณกรณีที่เป็นค่า Error ไปยังระบบ OMS เพื่อสร้างเหตุการณ์ไฟฟ้าขัดข้อง โดยที่ไม่เกิดเหตุการณ์ (การเชื่อมโยงระบบ)
GA14 : ระบบ GIS ข้อมูล วงจรระบบไฟฟ้า ไม่เป็นปัจจุบัน เนื่องจากต้องใช้บุคลากรและเวลา ในการดาเนินงาน
GB15 : การบันทึกข้อมูล อุปกรณ์บนระบบ GIS ไม่ถูกต้อง ไม่ครบถ้วน
GB16 : แนวทางการกรอก ข้อมูลในระบบ GIS แต่ละเขตยังไม่เป็นมาตรฐาน
GB7 : ผู้ใช้ไฟฟ้าพบเหตุไฟฟ้าขัดข้อง อาทิ หม้อแปลงระเบิด ตามเส้นทางการเดินทาง ทาให้ Agent ใช้เวลาในการตรวจสอบพื้นที่
GA9 : ระบบ SCADA ส่งข้อมูล ไปยังระบบ OMS เป็นคนละช่วงเวลากับ ระบบ GIS ส่งผลให้ข้อมูลทั้ง 3 ระบบข้อมูลไม่สอดคล้องกัน
GA10 : ระบบ SCADA จัดทำฐานข้อมูลเพื่อส่งข้อมูลสถานะอุปกรณ์ไปยังระบบ OMS ยังเป็นแบบ Manual (ทุกวันที่ 15 ของเดือน)
GB1 : E/O update ข้อมูลในระบบ OMS อาทิ สถานะเหตุการณ์ สถานะ การแก้ไขไฟฟ้า เวลาจ่ายไฟกลับคืน ล่าช้าในบางเหตุการณ์เนื่องจากต้อง ใช้เวลาในการตรวจสอบสภาพหน้างาน
GA1 : ระบบ OMS มีขั้นตอน การกรอกข้อมูลมีหลายขั้นตอน
GA2 : ระบบ OMS มีการปรับปรุงฐานข้อมูลระบบไฟฟ้า ใช้คนและระยะเวลาในการดำเนินงาน
GA3 : ระบบ OMS ไม่สามารถ บันทึกและส่งข้อมูลผู้ใช้ไฟกับระบบเกี่ยวข้องได้ ในกรณีปิดระบบ OMS เพื่อบำรุงรักษา
GA4 : ระบบ OMS ยังไม่ สามารถ เชื่อมโยงข้อมูลเหตุการณ์ ไฟฟ้าขัด ข้องกับระบบ Mobile Work Force
GB2 : ชุดปฏิบัติการระบบไฟฟ้า (ชุดแก้ไฟ) รายงานข้อมูลผ่านวิทยุบางพื้นที่ สัญญาญไม่ครอบคลุม ส่งผลให้การแจ้ง ข้อมูลกลับล่าช้า (Non Real-time Data)
GD1 : ผู้ใช้ไฟฟ้าไม่ทราบข้อมูลเลขที่ CA ทาให้ Agent ใช้เวลาในการตรวจสอบพื้นที่
