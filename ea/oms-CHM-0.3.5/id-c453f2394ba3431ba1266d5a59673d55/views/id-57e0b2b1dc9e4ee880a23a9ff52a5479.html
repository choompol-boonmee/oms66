3.5 ปรับปรุงโปรเซส ()
3.5.1 ปรับปรุงกระบวนงานให้มีการสลับจอการทำงานน้อยลง โดยการเชื่อมโยงระบบมากขึ้น
3.5.2 ลดการบันทึกข้อมูลของ E/O ลงโดยการให้หน้างานบันทึกข้อมูลโดยตรงมากขึ้น
3.5.3 เจ้าหน้าที่หน้างานได้เข้าถึงข้อมูลมากขึ้นผ่านอุปกรณ์โมบายทำให้ปฏิบัติงานได้มีประสิทธิภาพ
3.5 ผลการศึกษาวิเคราะห์ การปรับปรุงกระบวนการทำงาน ของระบบบริหารไฟฟ้าขัดข้อง
3.5.4 ลดภาระงานในการจัดทำรายงานที่เกี่ยวข้องกับการบริหารไฟฟ้าขัดข้อง
