สถาปัตยกรรมระบบคอมพิวเตอร์แม่ข่าย ()
2.2 สถาปัตยกรรมระบบคอมพิวเตอร์แม่ข่าย โครงสร้างพื้นฐานเครื่องคอมพิวเตอร์แม่ข่าย ต้องออกแบบในลักษณะที่ประกอบด้วย ชุดเครื่องคอมพิวเตอร์แม่ข่ายเสมือน (Set of Virtual Machines) ถูกจัดสรรภายใต้ ชุดของเครื่องคอมพิวเตอร์จริงทางกายภาพ (Several Physical Servers) ออกแบบให้เชื่อมต่อเข้าด้วยกันด้วยระบบเครือข่ายแลนเสมือน (VLAN) ที่ออกแบบมาแบบรีดันแดนซ์ (Redundant Virtual Local Area Network) เพื่อให้สามารถทำงานอย่างมีเสถียรภาพและมีความมั่นคงปลอดภัย และสามารถโอนถ่ายการทำงานไปยังศูนย์คอมพิวเตอร์สำรองได้อย่างมีประสิทธิภาพ ภาพต่อไปนี้แสดงตัวอย่างการออกแบบระบบคอมพิวเตอร์ให้สามารถรองรับความต้องการดังกล่าว โครงสร้างสถาปัตยกรรมของระบบ ต้องออกแบบให้แยกเป็นส่วนๆ ตามวัตถุประสงค์ของการใช้งาน ประกอบด้วย องค์ประกอบสำคัญดังต่อไปนี้
