ฟังก์ชันด้านการบริหารงานแก้ไขไฟฟ้าขัดข้อง (Outage Work Management ()
(2) ฟังก์ชันด้านการบริหารงานแก้ไขไฟฟ้าขัดข้อง (Outage Work Management) สนับสนุนการสั่งการแก้ไขไฟฟ้าขัดข้อง ตั้งแต่ การค้นหาสาเหตุไฟฟ้าขัดข้อง การจัดการบุคลากรแก้ไขไฟฟ้าขัดข้อง การเบิกและเตรียมอุปกรณ์เพื่อแก้ไขไฟฟ้าขัดข้อง การรือถอนหรือเปลี่ยนแปลงอุปกรณ์ การตัดจ่ายกระแสไฟฟ้าก่อนและหลังการแก้ไขระบบไฟฟ้า การจ่ายไฟกลับคืน การประมาณการเวลาที่ใช้ในการดำเนินการแก้ไข เป็นต้น
