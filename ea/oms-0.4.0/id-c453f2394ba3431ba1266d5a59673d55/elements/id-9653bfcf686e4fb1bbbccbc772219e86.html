7.26. สามารถนำเข้าข้อมูลตั้งต้นพร้อมตรวจสอบความถูกต้องจากระบบงานที่เกี่ยวข้อง ตามที่ กฟภ. กำหนด อย่างน้อยดังนี้ 7.26.1. ระบบ GIS แหล่งจ่ายไฟ EGAT, ระบบสายส่ง (HV), สถานีไฟฟ้า (Substation), ระบบจำหน่ายแรงกลาง (MV), ระบบจำหน่ายแรงต่ำ (LV) 7.26.2. ระบบข้อมูลลูกค้าข้อมูลมิเตอร์และระบบข้อมูลพนักงาน อย่างน้อยดังนี้ • ชื่อลูกค้า • หมายเลขมิเตอร์ • หมายเลขลูกค้า • ชื่อพนักงาน • ตำแหน่งพนังงาน • ทักษะการทำงาน 7.26.3. ระบบ SCADA ข้อมูลอุปกรณ์ เช่น Site ID หรือ Location ID ค่าแรงดัน ค่ากระแส เป็นต้น
