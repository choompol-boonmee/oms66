3.5.3 เจ้าหน้าที่หน้างานได้เข้าถึงข้อมูลมากขึ้นผ่านอุปกรณ์โมบายทำให้ปฏิบัติงานได้มีประสิทธิภาพ ()
ปัจจุบันเจ้าหน้าที่แก้ไฟหน้างานยังขาดเครื่องมือโมบาย รวมถึงซอฟต์แวร์สนับสนุนที่มีประสิทธิภาพเท่าที่ควร จึงทำให้ต้องเตรียมข้อมูลไว้ล่วงหน้าก่อนไปถึงหน้างาน โดยอาจใช้การคาดการณ์โดยใช้ประสบการณ์ และใช้ข้อมูลเก่าที่มีอยู่ไปก่อน ซึ่งอาจผิดพลาด คลาดเคลื่อน ไม่เป็นปัจจุบัน มีช่องทางในการสื่อสารกับผู้ใช้ไฟที่ได้รับผลกระทบไม่ดีเท่าที่ควร ถ้าเจ้าหน้าที่หน้างานได้เข้าถึงข้อมูลที่มีความถูกต้องและปัจจุบันมากขึ้นผ่านอุปกรณ์โมบาย ก็จะทำให้ขั้นตอนการทำงานลดลง และสามารถปรับปรุงกระบวนงานเพื่อลดภาระงานดังกล่าวได้
