7.12. สามารถส่งข้อมูลการแจ้งเตือนไปยัง ระบบที่เกี่ยวข้อง เช่น กรณีที่ระบบบริหารไฟฟ้าขัดข้อง (OMS) มีข้อมูลการทำงานของอุปกรณ์สูงผิดปกติ (Condition Base Maintenance) และสามารถติดตามข้อมูลแผนงาน หรือใบสั่งงานบำรุงรักษาจากใบแจ้งเตือนได้ อย่างน้อยดังนี้ 7.12.1. กิจกรรมที่ทำในแต่ละอุปกรณ์ 7.12.2. ระยะเวลาที่ใช้ในการทำงาน 7.12.3. หน่วยงานรับผิดชอบของแต่ละกิจกรรม
