การทำประกันภัย ()
29. การทำประกันภัย 29.1 ผู้รับจ้างต้องทำประกันภัยของระบบคอมพิวเตอร์ที่เสนอทั้งหมดในส่วนที่เป็นฮาร์ดแวร์โดยระบุผู้รับผลประโยชน์เป็นการไฟฟ้าส่วนภูมิภาคและส่งมอบต้นฉบับกรมธรรม์ประกันภัยพร้อมหลักฐานการชำระเบี้ยประกันภัยให้แก่การไฟฟ้าส่วนภูมิภาค ทั้งนี้ ให้ผลคุ้มครองภัยทุกชนิด เช่น ฟ้าผ่า อัคคีภัย อุทกภัย แผ่นดินไหว เป็นต้น ตลอดระยะเวลาโครงการ 29.2 ผู้รับจ้างต้องทำประกันภัยอันตราย แก่ผู้ปฏิบัติงานของผู้ว่าจ้างและบุคคลที่ 3 ในวงเงินเต็มมูลค่างานมีผลตลอดระยะเวลาทำงานจ้างตามสัญญาจนกว่าผู้ว่าจ้างจะรับมอบงาน (การไฟฟ้าส่วนภูมิภาคได้ออกหนังสือ PAC ตามข้อ 30 ให้แก่ผู้รับจ้างแล้ว) ทั้งนี้ในระหว่างที่ผู้รับจ้างรับผิดชอบ หากระบบบริหารไฟฟ้าขัดข้อง (OMS) เกิดความชำรุดเสียหายจนเป็นเหตุให้เกิดความเสียหายแก่ทรัพย์สินของการไฟฟ้าส่วนภูมิภาคและ/หรือบุคคลใดโดยไม่ได้เกิดจากความผิดของการไฟฟ้าส่วนภูมิภาค ผู้รับจ้างต้องรับผิดชอบต่อการไฟฟ้าส่วนภูมิภาคและ/หรือบุคคลใดต่อความเสียหายที่เกิดขึ้นทั้งสิ้น
