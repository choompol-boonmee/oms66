มีการกำหนดแบบจำลองข้อมูลร่วม (Common Data Model) เพื่ออธิบายนิยามชื่อ/ความหมายข้อมูล และรูปแบบข้อมูลกลาง (Common Format) ()
1. การที่ระบบบริหารไฟฟ้าขัดข้อง OMS จะสามารถสนับสนุนการบริหารไฟฟ้าขัดข้องได้อย่างมีประสิทธิภาพนั้น จำเป็นต้องอาศัยการเชื่อมโยงข้อมูลกับระบบอื่น การรับส่งและใช้ข้อมูลร่วมกันระหว่างซอฟต์แวร์ที่แตกต่างกันจำเป็นต้องมีการกำหนดแบบจำลองข้อมูลร่วม (Common Data Model) เพื่ออธิบายนิยามชื่อ/ความหมายข้อมูล และรูปแบบข้อมูลกลาง (Common Format) เพื่อให้ระบบซอฟต์แวร์ที่แตกต่างกันสามารถรับส่งข้อมูลและเข้าใจข้อมูลได้อย่างถูกต้องและมีคุณภาพ
