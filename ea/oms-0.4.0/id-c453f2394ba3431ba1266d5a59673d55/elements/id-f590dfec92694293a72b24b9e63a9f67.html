บันทึกและติดตามการดำเนินงานของพนักงาน ()
2.9. สามารถบันทึกและติดตามการดำเนินงานของพนักงานรายบุคคลหรือกลุ่มพนักงานที่เกี่ยวข้องกับงานแก้กระแสไฟฟ้าขัดข้อง อย่างน้อยดังนี้ • การยืนยันเวลาปฏิบัติงาน (Time Confirmations) ได้แก่ เวลาเริ่มต้น เวลาสิ้นสุด ระยะเวลาทำงาน • ทักษะ และความชำนาญของพนักงาน • การดำเนินงาน/กิจกรรมที่ทำ
