การเชื่อมโยงข้อมูลกับระบบ SCADA ()
6.2. การเชื่อมโยงข้อมูลกับระบบ SCADA เพื่อให้ได้สถานะของอุปกรณ์ในโครงข่ายระบบไฟฟ้าที่ส่งผลให้เกิดเหตุการณ์ไฟฟ้าขัดข้อง หรือส่งผลกระทบต่อการดำเนินการแก้ไขไฟฟ้าขัดข้อง CIM profile: http://pea.co.th/cim/profile/SCADA_OMS# รับส่งข้อมูลได้แบบรีลไทม์ (real time) ร้องขอพร้อมกันได้ไม่น้อยกว่า 12,000 รายการ รายละเอียดการเชื่อมโยง ตามภาคผนวก A2
