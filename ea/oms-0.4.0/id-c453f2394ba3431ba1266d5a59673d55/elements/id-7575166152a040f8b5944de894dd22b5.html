3.19. สามารถทำสำเนา (copy) รายการข้อมูลที่เคยบันทึกไว้แล้วในอดีตในลักษณะ ชุดข้อมูล (template) เพื่อนำกลับมาใช้ใหม่ได้อย่างมีประสิทธิภาพ เพื่อประหยัดเวลาการบันทึกโดยไม่ต้องเริ่มต้นบันทึกข้อมูลทั้งหมดตั้งแต่เริ่มต้น หรือ บันทึกด้วยมือทั้งหมด
