วัตถุประสงค์ ()
2. วัตถุประสงค์ 2.1 เพื่อจัดหา พัฒนา และติดตั้งระบบบริหารไฟฟ้าขัดข้อง (OMS) เพื่อใช้งานบริหารเหตุการณ์ไฟฟ้าขัดข้องทดแทนระบบเดิมที่จะสิ้นสุดสัญญา 2.2 เพื่อยกระดับและเพิ่มประสิทธิภาพการทำงาน โดยบูรณาการเชื่อมโยงระบบงานที่เกี่ยวข้องเพื่อให้มีความสะดวกรวดเร็ว ลดขั้นตอน และลดระยะเวลาในการปฏิบัติงาน 2.3 เพื่อเพิ่มประสิทธิภาพของระบบและการปฏิบัติงานให้สามารถให้บริการตอบสนองความต้องการ ของลูกค้าให้เกิดความพึงพอใจทั้งด้านคุณภาพและบริการ
