ศูนย์ข้อมูลกลางสำนักงานใหญ่การไฟฟ้าส่วนภูมิภาค และศูนย์ข้อมูลสำรอง (IDC DR) งวดที่ 1 ผู้รับจะต้องดำเนินการทบทวนสอบทานความต้องการร่วมกับผู้รับผิดชอบและออกแบบระบบ ตาม ๑.๑(๑) และ ส่งมอบ ดังต่อไปนี้ 1. รายงานผลการทบทวนความต้องการและออกแบบระบบ 2. สถาปัตยกรรมองค์กรของระบบบริหารไฟฟ้าขัดข้องในปัจจุบัน (AS-IS) 3. สถาปัตยกรรมองค์กรของระบบบริหารไฟฟ้าขัดข้องเป้าหมาย (TO-BE) 4. แผนการติดตั้งฮาร์ดแวร์ของระบบคอมพิวเตอร์แม่ข่าย (Computer Server Installation Plan) 5. แผนการพัฒนาระบบซอฟต์แวร์ระบบบริหารไฟฟ้าขัดข้อง (OMS Software Development Plan) 6. Development Plan) 9. แผนการนำเข้าข้อมูล (Data Migration Plan) 10. แผนการฝึกอบรม On-the-Job-Training (OJT) 11. แผนการนำระบบบริหารไฟฟ้าขัดข้องออกใช้งาน 12. แผนการฝึกอบรมบุคลากร
