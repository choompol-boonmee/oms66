การเชื่อมโยงข้อมูลโครงข่ายไฟฟ้ากับระบบ GIS ()
6.1. การเชื่อมโยงข้อมูลโครงข่ายไฟฟ้ากับระบบ GIS เพื่อให้ได้มาซึ่งข้อมูลโครงข่ายไฟฟ้าที่ถูกต้อง ครบถ้วน และเป็นปัจจุบัน และสามารถส่งข้อมูลโครงข่ายไฟฟ้าที่มีการแก้ไขปรับปรุงไประบบบริหารไฟฟ้าขัดข้อง (OMS) ได้ ในการนี้อาจจำเป็นต้องใช้ซอฟต์แวร์เครื่องมือเพื่อการบริหารจัดการหรือแปลงข้อมูลโครงข่ายไฟฟ้า (Grid Model Management Tool) CIM profile: http://pea.co.th/cim/profile/GridGIS_OMS# รับส่งข้อมูลได้แบบแบทช์ (batch) และ แบบรีลไทม์ (real time) ร้องขอพร้อมกันได้ไม่น้อยกว่า 7,000 รายการ รายละเอียดการเชื่อมโยง ตามภาคผนวก A1
