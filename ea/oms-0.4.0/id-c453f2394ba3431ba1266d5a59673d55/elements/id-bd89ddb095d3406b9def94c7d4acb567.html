วิธีการและราคากลาง ()
การไฟฟ้าส่วนภูมิภาคมีความประสงค์จะประกวดราคาจ้างจัดหา พัฒนา และติดตั้งระบบบริหารไฟฟ้าขัดข้อง (Outage Management System: OMS) ด้วยวิธีประกวดราคาอิเล็กทรอนิกส์ ราคากลางของงานจ้างในการประกวดราคาครั้งนี้เป็นเงินทั้งสิ้น 1,018,849,720.- บาท (หนึ่งพันสิบแปดล้านแปดแสนสี่หมื่นเก้าพันเจ็ดร้อยยี่สิบบาทถ้วน)
