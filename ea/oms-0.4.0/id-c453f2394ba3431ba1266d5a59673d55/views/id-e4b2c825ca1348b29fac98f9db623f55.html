Book 1 Overview สถาปัตยกรรมโครงสร้างพื้นฐานและการเชื่อมโยง
ภาพรวมโครงการ การไฟฟ้าส่วนภูมิภาค
ภาพรวมโครงการ ภาพรวมการให้บริการกระแสไฟฟ้า
ภาพรวมโครงการ โครงสร้างพื้นฐานที่เกี่ยวข้อง
ภาพรวมโครงการ กระบวนงานบริหารไฟฟ้าขัดข้อง
สถาปัตยกรรมโครงสร้างพื้นฐานและการเชื่อมโยง การออกแบบระบบเพื่อความต่อเนื่องในการทำงาน
สถาปัตยกรรมโครงสร้างพื้นฐานและการเชื่อมโยง สถาปัตยกรรมระบบคอมพิวเตอร์แม่ข่าย
สถาปัตยกรรมระบบคอมพิวเตอร์แม่ข่าย ส่วนใช้งานจริง หรือ Production Environment (PDE)
สถาปัตยกรรมระบบคอมพิวเตอร์แม่ข่าย ส่วนเตรียมการ หรือ Pre-Production Environment (PPE)
สถาปัตยกรรมระบบคอมพิวเตอร์แม่ข่าย ส่วนเข้าถึงได้จากภายนอก หรือ DMZ Environment (DMZE)
สถาปัตยกรรมระบบคอมพิวเตอร์แม่ข่าย ส่วนบริหารจัดการ หรือ System Management Environment (SME)
สถาปัตยกรรมระบบคอมพิวเตอร์แม่ข่าย ส่วนจำลองสถานการณ์ หรือ Simulation Environment (SIE)
