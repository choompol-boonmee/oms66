02-AMI-screen ()
ภาพต่อไปนี้แสดงตัวอย่างหน้าจอระบบแก้ไขไฟฟ้าขัดข้อง ที่มีความสามารถในการเชื่อมโยงข้อมูลกับระบบ AMI และสามารถรับทราบข้อมูลได้ตลอดเวลาว่า มิเตอร์ตัวไหนก็ไฟฟ้าขัดข้องขึ้นบ้าง และถ้ามีการแก้ไขไฟฟ้าขัดข้องเสร็จสิ้นแล้ว ระบบก็จะรู้ได้ทันทีว่าปัญหาได้รับการแก้ไขแล้ว เนื่องจาก มิเตอร์ได้ส่งข้อมูลมาบอกระบบส่วนกลางของ AMI ทำให้ระบบไฟฟ้าขัดข้องสามารถเข้าใจสถานการณ์ได้อย่างอัตโนมัติ
