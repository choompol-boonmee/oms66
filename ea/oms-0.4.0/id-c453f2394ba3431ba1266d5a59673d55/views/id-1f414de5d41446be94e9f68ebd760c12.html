เอกสารแนบท้ายเอกสารประกวดราคาอิเล็กทรอนิกส์ รายละเอียดและขอบเขตของงาน
เอกสารแนบท้ายเอกสารประกวดราคาอิเล็กทรอนิกส์ แบบบัญชีเอกสารที่กำหนดไว้ในระบบจัดซื้อจัดจ้างภาครัฐด้วยอิเล็กทรอนิกส์ (1) บัญชีเอกสารส่วนที่ 1 (2) บัญชีเอกสารส่วนที่ 2
รายละเอียดและขอบเขตของงาน เงื่อนไขเฉพาะงาน
รายละเอียดและขอบเขตของงาน ขอบเขตรายละเอียดของงาน (TOR)
รายละเอียดและขอบเขตของงาน ตาราง Table of Compliance
รายละเอียดและขอบเขตของงาน รายละเอียดขั้นตอนการดำเนินการสาธิต และหลักเกณฑ์การพิจารณาในส่วนของเกณฑ์ราคาประกอบเกณฑ์อื่น
รายละเอียดและขอบเขตของงาน แบบฟอร์มรายละเอียดการเสนอราคา (Price schedule)
รายละเอียดและขอบเขตของงาน แบบฟอร์มเอกสาร แบบฟอร์ม 1: รายละเอียดทีมงาน แบบฟอร์ม 2 : บัญชีแสดงรายละเอียดราคางานบำรุงรักษาระบบบริหารไฟฟ้าขัดข้อง (OMS) หลังหมดระยะเวลารับประกัน
