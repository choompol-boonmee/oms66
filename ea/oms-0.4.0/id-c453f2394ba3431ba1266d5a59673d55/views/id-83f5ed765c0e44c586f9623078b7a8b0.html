ข้อกำหนดความต้องการเชื่อมโยงข้อมูล (System Integration)
มีการกำหนดแบบจำลองข้อมูลร่วม (Common Data Model) เพื่ออธิบายนิยามชื่อ/ความหมายข้อมูล และรูปแบบข้อมูลกลาง (Common Format)
แบบจำลองข้อมูลที่เป็นมาตรฐาน (Standard Data Model) และรูปแบบข้อมูลมาตรฐาน (Standard data format)
มาตรฐานข้อมูลไออีซีซิม (IEC Common Information Model : CIM)
หลักการ ‘ข้อมูลต้องมีที่มาจากแหล่งข้อมูลเดียว’ หรือ ‘Single Source of Truth’ เป็นหลักการสำคัญที่ กฟภ. ใช้ในการรักษาคุณภาพ ความถูกต้อง และความน่าเชื่อถือของข้อมูล
ผู้รับจ้างต้องรับผิดชอบค่าใช้จ่ายในการพัฒนาระบบเชื่อมโยงทั้งหมด ในฝั่งที่เกี่ยวข้องกับระบบบริหารไฟฟ้าขัดข้อง (OMS) ที่เสนอ
