ระบบต้องรองรับข้อมูลลูกค้าได้อย่างน้อย 25,000,000 ราย
หน้าจอใช้งานระบบต้องแสดงผลทุกหน้าจอทั้งหมดด้วยภาษาไทย และอังกฤษได้
ระบบต้องทางานผ่าน Web Browser และหาก Web Browser มีการ Update Version ระบบต้องสามารถทางานได้ตาม Version ล่าสุดในขณะนั้นได้
ในการเชื่อมโยง (Integration) และการเชื่อมต่อ (Interface) ต่างๆ ต้องสามารถตรวจสอบความถูกต้องและข้อผิดพลาดของข้อมูลที่รับส่งระหว่างระบบ ได้
สามารถ Upgrade ระบบโดยไม่ทำให้เกิด Down Time และไม่รบกวนการทำงานของระบบหรือการเชื่อมต่อ (Integration) กับระบบอื่นๆ ภายนอก
การปรับปรุงข้อมูลจากระบบอื่นๆที่เชื่อมโยงกับระบบบริหารไฟฟ้าขัดข้องต้องปรับปรุงเฉพาะข้อมูลที่เปลี่ยนแปลงแบบอัตโนมัติ โดยไม่ส่งผลกระทบต่อผู้ใช้งานของระบบ
นาเข้าข้อมูลตั้งต้นพร้อมตรวจสอบความถูกต้องจากระบบงานที่เกี่ยวข้อง ตามที่ กฟภ. กาหนด เช่น - ระบบ GIS แหล่งจ่ายไฟ EGAT, ระบบสายส่ง (HV), สถานีไฟฟ้า (Substation), ระบบจาหน่ายแรงกลาง (MV), ระบบจาหน่ายแรงต่า (LV) - ระบบ CIS ข้อมูลลูกค้า ข้อมูลมิเตอร์ เช่น ชื่อ หมายเลขมิเตอร์ หมายเลขลูกค้า เป็นต้น- ระบบ HR ข้อมูลพนักงาน เช่น ชื่อ ตาแหน่ง ทักษะการทางาน เป็นต้น- ระบบ SCADA ข้อมูลอุปกรณ์ เช่น Site ID หรือ Location ID ค่าแรงดัน ค่ากระแส เป็นต้น
