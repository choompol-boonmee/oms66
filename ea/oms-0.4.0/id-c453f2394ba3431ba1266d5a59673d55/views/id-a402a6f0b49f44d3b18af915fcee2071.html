01-phase-predict ()
ปัจจุบันเทคโนโลยีปัญญาประดิษฐ์ (AI: Artificial Intelligent) มีก้าวหน้ามาก โดยเฉพาะเทคโนโลยีด้าน แมชชีนเลิร์นนิ่ง (Machine Learning) จึงมีการประยุกต์ใช้เทคโนโลยีปัญญาประดิษฐ์นี้ในการคาดการณ์ต่างๆ ได้แม่นยำ ภาพข้างล่างต่อไปนี้แสดงตัวอย่างการคาดการณ์เฟสว่ามิเตอร์ที่เชื่อมโยงกับหม้อแปลงจำหน่าย ทำให้ถึงแม้จะไม่มีข้อมูลเฟสบันทึกไว้ หรือข้อมูลเฟสไม่ถูกต้อง แต่ด้วยการใช้ปัญญาประดิษฐ์ดังกล่าว ทำให้สามารถคาดการณ์เฟสของมิเตอร์แต่ละตัวได้ นอกจากนี้ยังมีการประยุกต์ใช้ปัญญาประดิษฐ์ในกรณีอื่นๆ อีกมากมาย
