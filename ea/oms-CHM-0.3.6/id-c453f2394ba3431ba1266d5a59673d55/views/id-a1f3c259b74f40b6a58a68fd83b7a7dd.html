C6.3.3.1 งานบริหารจัดการ เหตุการณ์ใน ระบบจำหนายไฟฟ้า สาเหตุภายในของ PEA
C6.3.3.1-9 Field Crew รับงาน ออกเดินทางค้นหา สาเหตุไฟฟ้าขัดข้อง และออกเดินทาค้นหาสาเหตุ เมื่อพบสาเหตุ ตรวจสอบ รายงานสาเหตุ สภาพหน้างาน พิจารณาแนวทางแก้ไข และประมาณการ ระยะเวลาแก้ไข หรือ ประมาณการเวลาจ่ายไฟกลับคืน
C6.3.3.1-10 มอบหมายใบสั่งงานเพื่อให้ ชุดปฏิบัติการ ระบบไฟฟ้า เข้าตรวจสอบ และค้นหาสาเหตุ
C6.3.3.1-11 แจ้งเหตุการณ์ไฟฟ้า ขัดข้องให้ผู้ใช้ไฟทราบ
C6.3.3.1-12 รับทราบเหตุการณ์ สภาพหน้างาน (เช่น สาเหตุ และระยะเวลาจ่ายไฟกลับคืน)
C6.3.3.1-13 บันทึกข้อมูล เหตุการณ์ สภาพหน้างาน (เช่น สาเหตุ และระยะเวลาจ่ายไฟกลับคืน)
C6.3.3.1-14 บันทึกข้อมูล เหตุการณ์ สภาพหน้างาน (เช่น สาเหตุ และระยะเวลาจ่ายไฟกลับคืน)
