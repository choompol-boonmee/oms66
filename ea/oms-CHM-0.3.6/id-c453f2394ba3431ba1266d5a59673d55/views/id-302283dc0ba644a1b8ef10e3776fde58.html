16-JD-กองปฏิบัติ ()
JD - กองปฏิบัติการ
1. วิเคราะห์ วางแผน ออกแบบ ปรับปรุงการจ่ายไฟ ในระบบสายส่งและระบบจำหน่าย
4. สั่งการ และสั่งงานหน่วยปฏิบัติงานแก้ไฟฟ้าขัดข้อง และบันทึกเหตุการณ์ไฟฟ้าขัดข้องในระบบสายส่ง สถานีไฟฟ้าและระบบจำหน่ายแรงสูง
7. ร่วมบูรณาการและดำเนินการตามแนวทางการบริหารจัดการทรัพย์สิน (AssetManagement) ของ กฟภ.
3. งานปฏิบัติการ (Operate) อุปกรณ์ในสายส่ง สถานีไฟฟ้าและระบบจำหน่ายแรงสูงโดยระบบ SCADA
5. จัดการ กำกับดูแล การปฏิบัติงานของพนักงานประจำสถานีไฟฟ้า สถานีไฟฟ้าพลังนํ้าและพลังงานทดแทนหน่วยปฏิบัติงานสถานีไฟฟ้า และหน่วยจัดการงานสถานีไฟฟ้า
2. วิเคราะห์ความเชื่อถือได้ของระบบไฟฟ้า
6. บริหารจัดการระบบศูนย์ควบคุมการจ่ายไฟฟ้า (SCADA/DMS) และพัฒนาApplication Software ที่เกี่ยวข้อง
