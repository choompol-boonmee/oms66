C6.3.3.1-13 ประเด็นปัญหาและสิ่งที่ต้องปรับปรุงในอนาคต 1. ชุดปฏิบัติการระบบไฟฟ้า (ชุดแก้ไฟ) รายงานข้อมูลผ่านวิทยุบางพื้นที่สัญญาญไม่ครอบคลุม ส่งผลให้การแจ้งข้อมูลกลับล่าช้า (NonReal-time Data) 2. พัสดุแก้ไฟไม่มีคุณภาพ และขาดแคลน 3. เครื่องมือ อุปกรณ์ความปลอดภัย ขาดแคลน 4. ขนาดรถแก้ไฟไม่เหมาะสมกับพื้นที่ 5. การติดตั้งอุปกรณ์ไม่เป็นมาตรฐาน
